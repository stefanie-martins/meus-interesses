CSS

CSS é uma sigla para Cascading Style Sheets, que em português significa Folhas de Estilo em Cascata. É uma linguagem de estilo utilizada para definir como elementos HTML devem ser exibidos em um documento web. O CSS permite controlar o layout, os núcleos, as fontes, as animações e outros aspectos visuais de um site, separando a apresentação do conteúdo, o que facilita a manutenção e personalização do design dos sites. Com o CSS, é possível criar estilos diferentes para diferentes dispositivos, adaptando o conteúdo para ser exibido de forma adequada em telas de computadores, tablets e smartphones.
