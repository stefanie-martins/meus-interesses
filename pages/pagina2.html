HTML

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada na construção de páginas e aplicativos para a web. É a base de toda a estrutura de um site, definindo a organização e estrutura do conteúdo, como texto, imagens, vídeos e links. O HTML usa tags para marcar elementos e atributos para definir características desses elementos. É uma linguagem fundamental para a criação de páginas web e é interpretada pelos navegadores para exibição correta do conteúdo.
